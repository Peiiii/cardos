## 文档规范

1. 所有文档使用 Markdown 格式
2. 文档命名使用小写字母，单词间用连字符（-）连接
3. 文档更新需要在更新记录中记录
4. 重要决策需要在相应文档中说明原因和考虑因素

## 如何贡献文档

1. 遵循现有的文档结构
2. 保持文档的清晰和简洁
3. 及时更新文档
4. 在更新记录中记录变更
